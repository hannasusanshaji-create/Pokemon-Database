##  Features

- **Pokédex View** – Browse Pokémon cards, search by name or ID (e.g. *pikachu*, *charizard*, or *25*).
- **Pokémon History** – Access extra background or lore 
- Clean, intuitive UI with a classic Pokémon theme.
